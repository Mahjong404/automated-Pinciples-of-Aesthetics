## 这是一个超星学习通上《美学原理》课程的自动刷课脚本

## 设置方法
### 请确保电脑中有Edge浏览器。该脚本目前只适用于Edge浏览器，如有需要，请手动替换为Chrome驱动及相应设置代码
### 请在data/accout.json文件中，填写您的账号和密码以及未完成的小节对应的视频页的网址
### 如果视频已完成，但该小节题目未完成，请仍然粘贴视频页的网址，然后在视频开始播放后手动拉动进度条到末尾
### 请挂在前台播放视频，视频开始播放后请勿将鼠标移出网页外（没做虚拟鼠标），如播放停止，请手动开始播放

该脚本基于Python及其Selenium库实现，如果觉得不错的，请点个star吧，谢谢！
